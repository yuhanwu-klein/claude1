# 角色五：暗夜王子

## 基本信息
- **角色类型**：中世纪王子
- **艺术风格**：简约优雅，年轻活力

## 外观描述
- **面部**：英俊的脸庞，棱角分明，带着一丝叛逆的神情
- **发型**：飘逸的黑色长发，部分编成小辫
- **服装**：
  - 深海军蓝色丝绒外套，金色刺绣边缘
  - 黑色紧身内衬，显出修长身材
  - 深红色腰封，镶嵌小宝石
  - 黑色高筒皮靴
  - 披着深色披风，内衬为银色丝绸
  - 佩戴细剑，剑柄精致华美

## 姿态
- 站立姿态，略微侧身
- 一手搭在剑柄上，显示自信
- 眼神桀骜不驯，带着年轻的傲气
- 嘴角微微上扬

## 背景设定
- **场景**：城堡宴会厅
- **色调**：深蓝色、黑色、金色
- **细节**：
  - 背景是华丽的拱形天花板，绘有星空壁画
  - 墙上挂着深色挂毯，金线刺绣家族徽章
  - 巨大的水晶吊灯悬挂，烛光闪烁
  - 长桌上摆放着银质餐具和高脚酒杯
  - 地面是黑白相间的大理石棋盘格
  - 窗外是夜色中的花园，月光洒落
  - 空气中飘着香料酒和烤肉的香气
